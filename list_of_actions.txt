From hamachi README
Quick Start
	1. Run 'hamachi login' to set the daemon online and to create an account.
	2. Optionally, run 'hamachi set-nick <nickname>'. Unless you select a
	   nickname, other Hamachi peers will see your client name as blank.
	3. Run 'hamachi join <network id>' to join a network.
	4. Run 'hamachi list' to list network members and their status.
